# 신입 개발자 기술 트렌드 분석

## 1. 프로젝트 제목
- 신입 개발자 기술 트렌드 분석 (사람인 채용 공고 데이터 기반)

## 2. 사용하는 데이터
- **수집 소스:** 사람인 (www.saramin.co.kr)
- **대상:** '신입 백엔드', '신입 프론트엔드' 등 직무별 채용 공고 약 1000건
- **핵심 정보:** 공고 제목, 자격 요건, 우대 사항 텍스트 데이터

## 3. 데이터 수집 방법
- **크롤링:** Python (requests, BeautifulSoup)을 활용하여 웹페이지 데이터 수집
- **저장:**
    - 1차: 수집된 원본 데이터를 JSON 파일로 백업
    - 2차: MongoDB에 정식으로 적재하여 관리 및 분석에 활용

## 4. 분석할 내용
- **핵심 기술 스택 분석 (정량 분석):**
    - 전체 및 직무별 Top 10 기술 스택 추출 (e.g., Java, Spring, React)
    - 기술 조합 분석: 함께 자주 사용되는 기술 페어 분석 (e.g., Spring Boot + JPA)
- **잠재 역량 분석 (정성 분석):**
    - 우대사항에 나타난 소프트 스킬 키워드 분석 (e.g., 협업, 성장, 커뮤니케이션)

## 5. 프로젝트 범위
- 데이터 수집 ~ DB 저장 자동화
- 데이터 전처리 및 정형화 (by Pandas)
- 데이터 분석 및 시각화 (by Matplotlib, Seaborn)
- 분석 결과 보고서(PPT) 작성

## 6. AI 활용
- **Code Generation (코드 생성):**
    - 크롤러, 데이터 전처리기, 시각화 등 Python 코드 초안 작성 및 디버깅
- **Analysis Idea (분석 아이디어):**
    - "수집된 데이터로 어떤 추가 분석을 할 수 있을까?"에 대한 아이디어 도출
- **Summarization (결과 요약):**
    - 분석된 그래프와 데이터를 해석하여 PPT에 기입할 핵심 인사이트 및 문장 생성
